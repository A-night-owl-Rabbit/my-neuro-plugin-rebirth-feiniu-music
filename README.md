# 重生之网易云没火肥牛要火了

网易云音乐 my-neuro 插件，让你的 Live2D 角色变身全能音乐助手。

搜歌、播放、歌单管理、智能推荐、偏好分析、随机播放，全部语音驱动，带完整 Live2D 联动（嘴型同步、歌词气泡、麦克风动作）。

## 快速开始（开箱即用）

### 1. 下载插件

将本仓库所有文件放入肥牛的插件目录：

```
my-neuro-main/live-2d/plugins/community/rebirth-feiniu-music/
```

> ncm-cli 及所有依赖已打包在 `node_modules/` 中，无需额外安装 Node.js 或 npm。

### 2. 申请 API Key

前往 [网易云音乐开放平台](https://developer.music.163.com) 注册并申请 API Key，获取 `appId` 和 `privateKey`。入驻成功后，在应用管理的应用详情中获取 API Key。

### 3. 在肥牛插件配置中填写凭证

在肥牛中打开本插件的**配置页**（`plugin_config` 对应界面），将开放平台提供的 **`appId`** 与 **`privateKey`** 填入对应配置项并保存。

- 两项需**同时填写**，插件会在启动与调用工具时自动同步到本机 `ncm-cli`。
- **不要**把真实密钥提交到公开 Git 仓库；仓库内的 `plugin_config.json` 仅作字段说明，请在本机配置界面填写。

### 4. 首次登录：让 AI 扫码登录

配置好 API 凭证后，**首次使用**或登录失效时，需要绑定网易云账号。请直接对角色说例如：

- 「**登录网易云**」
- 「**网易云扫码登录**」

AI 会调用 `netease_login` 工具，在窗口内弹出扫码界面，使用**网易云音乐 App** 扫码即可完成授权。部分场景下若播放失败且判定为未登录，插件也会尝试自动拉起扫码流程。

### 5. 启动使用

重启或启动肥牛后，对角色说「播放 xxx」等即可使用音乐功能。

## 主要功能

- **搜索播放** — 对角色说「播放 xxx」「搜一下 xxx」，即可搜索并播放歌曲
- **歌单播放** — 「播放我的歌单 xxx」，加载整个歌单到队列顺序或随机播放
- **随机播放** — 「随机播放」「打乱播放」，支持歌单和推荐的随机模式
- **每日推荐** — 「播放每日推荐」，获取并播放网易云每日推荐
- **智能推荐** — 三种模式：
  - **私人漫游** — 「推荐点喜欢的」，基于偏好自动推歌
  - **心动模式** — 「来点类似的歌」，基于当前播放歌曲推荐相似曲目
  - **雷达歌单** — 获取个性化雷达歌单
- **偏好分析** — 「分析我的音乐偏好」，分析红心歌单的曲风标签、常听艺人、时段分布
- **播放控制** — 暂停、继续、下一首、上一首、音量调节、跳转
- **歌单管理** — 查看、创建、添加/删除歌曲、搜索歌单
- **Live2D 联动** — 嘴型随音乐动、歌词气泡跟随模型、麦克风动作

## 歌词气泡与其它配置

在插件配置页面可调整 **API 凭证**、**歌词气泡**外观与位置等。歌词相关项修改后，一般在**下一首歌**播放时生效。

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 网易云开放平台 App ID | 同步至 ncm-cli 的 appId | （空，须填写） |
| 网易云开放平台 Private Key | 同步至 ncm-cli 的私钥 | （空，须填写） |
| 歌词气泡水平偏移 | 相对模型中心的水平偏移(px) | -20 |
| 歌词气泡垂直偏移 | 相对模型中心的垂直偏移(px) | -20 |
| 歌词气泡最小/最大宽度 | 气泡宽度(px) | 160 / 320 |
| 歌词气泡垂直/水平内边距 | 内边距(px) | 16 / 24 |
| 歌词气泡字体大小 | 文字大小(px) | 16 |
| 歌词气泡圆角 | 圆角半径(px) | 24 |
| 歌词气泡缩放 | 整体缩放比例，1.0 为原始大小 | 1.0 |

## 歌曲为什么播放不了？

插件通过 [网易云音乐开放平台](https://developer.music.163.com) API（`ncm-cli`）获取歌曲，不同于网易云音乐 App 直接播放，存在以下限制层级：

### 播放权限层级说明

| 层级 | 字段 | 含义 | 影响 |
|------|------|------|------|
| 1. 版权状态 | `privilege.st` | `st < 0` 表示歌曲已下架或无版权 | 任何方式都无法播放，App 中也显示灰色 |
| 2. 费用类型 | `fee` | `0`=免费, `1`=VIP, `4`=需购买专辑, `8`=非会员低音质/会员高音质 | 决定是否需要 VIP 或单独付费 |
| 3. 开放平台授权 | `playFlag` | 歌曲是否被授权在开放平台（第三方）播放 | **即使是黑胶 VIP，部分歌曲在开放平台上仍无授权** |

### "我是黑胶会员，为什么还是播放不了？"

这是最常见的疑问。核心原因：

1. **开放平台 ≠ App**：开放平台 API 有独立的版权授权范围，比 App 小。部分歌曲虽然在 App 里 VIP 可播，但版权方未授权在开放平台（第三方应用）中播放，API 返回 `playFlag=false`。

2. **插件的三级降级策略**：当原版不可播时，插件会自动尝试：
   - ① 通过 `ncm-cli play` 截获实际播放地址（可能绕过 playFlag 限制）
   - ② 搜索同名可播放版本（如不同歌手翻唱、Live 版等）
   - ③ 尝试公开外链和 enhance API
   - 全部失败才提示「不可播放」

3. **登录状态影响**：未登录时所有 VIP 歌曲（`fee=1`）都无法播放。登录过期后会退化为游客权限。插件启动后每 30 分钟检查登录状态，失效时自动提醒重新扫码。

### 各种报错对照

| 提示信息 | 原因 | 解决方法 |
|----------|------|----------|
| `(不可播放)` | 开放平台未授权该歌曲（`playFlag=false`） | 插件会自动搜索替代版本；如果是 VIP 歌曲，确保已登录 |
| `(VIP)` | 需要 VIP 会员才能播放（`fee=1`） | 确保使用 VIP 账号扫码登录 |
| `暂无开放平台播放权限（App可播放但API未授权）` | 三种取链方式均失败 | 该歌曲确实无法通过开放平台播放，尝试搜索其他版本 |
| `搜索结果均无法播放（可能需要VIP权限）` | 所有候选歌曲都取链失败 | 检查登录状态，确认 VIP 是否生效 |
| `原版和替代版均不可用` | 队列播放时原版及替代版都无法获取音频 | 自动跳过该歌曲，播放下一首 |

## 常见问题

| 问题 | 解决方法 |
|------|----------|
| 提示未登录或无法搜索/播放 | 对角色说「登录网易云」完成扫码；确认已填写 appId 与 privateKey |
| 歌曲不可播放 | 见上方「歌曲为什么播放不了？」章节 |
| 每天都要重新扫码登录 | 开放平台登录令牌有效期有限，建议升级 ncm-cli 到最新版（`ncm-cli upgrade` 或 `npm i -g @music163/ncm-cli@latest`）。插件会在登录失效时自动拉起扫码 |
| 歌词气泡配置不生效 | 修改配置后播放下一首歌时生效 |
| 心动模式失败 | 需要先播放一首歌，心动模式基于当前歌曲推荐 |
| 登录过期 | 再次对角色说「登录网易云」触发 `netease_login`；插件每 30 分钟自动检查登录状态 |
