# 重生之网易云没火肥牛要火了

网易云音乐 my-neuro 插件，让你的 Live2D 角色变身全能音乐助手。

搜歌、播放、歌单管理、智能推荐、偏好分析、随机播放，全部语音驱动，带完整 Live2D 联动（嘴型同步、歌词气泡、麦克风动作）。

## 快速开始（开箱即用）

### 1. 下载插件

将本仓库所有文件放入肥牛的插件目录：

```
my-neuro-main/live-2d/plugins/community/rebirth-feiniu-music/
```

> ncm-cli 及所有依赖已打包在 `node_modules/` 中，无需额外安装 Node.js 或 npm。

### 2. 申请 API Key

前往 [网易云音乐开放平台](https://developer.music.163.com) 注册并申请 API Key，获取 `appId` 和 `privateKey`。

### 3. 双击 setup.bat

运行安装向导，按提示：
1. 粘贴 `appId`
2. 粘贴 `privateKey`（支持直接粘贴内容或输入密钥文件路径）
3. 用网易云音乐 App 扫码登录

### 4. 启动肥牛

重启肥牛应用，对角色说"播放xxx"即可听歌。

## 主要功能

- **搜索播放** — 对角色说"播放xxx"、"搜一下xxx"，即可搜索并播放歌曲
- **歌单播放** — "播放我的歌单xxx"，加载整个歌单到队列顺序或随机播放
- **随机播放** — "随机播放"、"打乱播放"，支持歌单和推荐的随机模式
- **每日推荐** — "播放每日推荐"，获取并播放网易云每日推荐
- **智能推荐** — 三种模式：
  - **私人漫游** — "推荐点喜欢的"，基于偏好自动推歌
  - **心动模式** — "来点类似的歌"，基于当前播放歌曲推荐相似曲目
  - **雷达歌单** — 获取个性化雷达歌单
- **偏好分析** — "分析我的音乐偏好"，分析红心歌单的曲风标签、常听艺人、时段分布
- **播放控制** — 暂停、继续、下一首、上一首、音量调节、跳转
- **歌单管理** — 查看、创建、添加/删除歌曲、搜索歌单
- **Live2D 联动** — 嘴型随音乐动、歌词气泡跟随模型、麦克风动作

## 歌词气泡配置

在插件配置页面可调整歌词气泡的外观和位置，修改后下一首歌播放时生效：

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 歌词气泡水平偏移 | 相对模型中心的水平偏移(px) | -20 |
| 歌词气泡垂直偏移 | 相对模型中心的垂直偏移(px) | -20 |
| 歌词气泡最小宽度 | 气泡最小宽度(px) | 160 |
| 歌词气泡最大宽度 | 气泡最大宽度(px) | 320 |
| 歌词气泡字体大小 | 歌词文字大小(px) | 16 |
| 歌词气泡圆角 | 气泡圆角半径(px) | 24 |
| 歌词气泡缩放 | 整体缩放比例，1.0 为原始大小 | 1.0 |

## 工具列表

插件提供 12 个 AI 工具：

| 工具 | 说明 |
|------|------|
| `netease_search` | 搜索歌曲/歌单/歌手 |
| `netease_play` | 搜索并播放歌曲 |
| `netease_play_playlist` | 播放整个歌单（支持随机） |
| `netease_control` | 播放控制（暂停/继续/停止/上下首/音量/跳转） |
| `netease_state` | 查看当前播放状态 |
| `netease_queue` | 播放队列管理 |
| `netease_recommend` | 每日推荐 |
| `netease_playlist` | 歌单管理（查看/创建/添加/删除/搜索） |
| `netease_smart_recommend` | 智能推荐（私人漫游/心动模式/雷达歌单） |
| `netease_preference` | 偏好分析 |
| `netease_shuffle` | 随机播放开关 |
| `netease_login` | 扫码登录 |

## 常见问题

| 问题 | 解决方法 |
|------|----------|
| setup.bat 提示找不到 ncm-cli | 确保 `node_modules` 目录完整下载 |
| 歌曲不可播放 | 部分歌曲需要 VIP 权限，插件会自动搜索可播放版本 |
| 歌词气泡配置不生效 | 修改配置后播放下一首歌时生效 |
| 心动模式失败 | 需要先播放一首歌，心动模式基于当前歌曲推荐 |
| 登录过期 | 对角色说"登录网易云"或重新运行 setup.bat |
